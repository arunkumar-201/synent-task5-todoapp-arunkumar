# 📌 Project Overview

A modern and responsive Todo Application built using HTML, CSS, and JavaScript.

This project helps users:

- ✅ Add Tasks
- ✅ Delete Tasks
- ✅ Mark Tasks as Completed
- ✅ Save Tasks using Local Storage
- ✅ Manage Daily Activities Easily

The UI is inspired by modern fintech applications like **PhonePe** with glassmorphism effects and gradient backgrounds.
